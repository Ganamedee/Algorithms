Sorting Algorithms

Sorting algorithms arrange elements in a specific order, typically in ascending or descending order. Each algorithm has its own approach and efficiency characteristics.

Algorithm Time Complexity (Avg) Space Complexity
Bubble Sort O(n²) O(1)
Selection Sort O(n²) O(1)
Insertion Sort O(n²) O(1)
Quick Sort O(n log n) O(log n)

Searching Algorithms

Searching algorithms are designed to find a particular element within a data structure. Each algorithm uses different strategies to locate elements efficiently.

Algorithm Time Complexity Space Complexity
Linear Search O(n) O(1)
Binary Search O(log n) O(1)

Note: Binary search requires the array to be sorted first.

Pathfinding Algorithms

Pathfinding algorithms find the shortest or optimal path between two points in a graph or grid. These algorithms are essential in navigation, gaming, and network routing.

Algorithm Time Complexity Space Complexity
A* (A-Star) O(b^d) O(b^d)
Dijkstra's O(V² + E) O(V)

Note: For A*, 'b' is the branching factor and 'd' is the depth. For Dijkstra's, 'V' is the number of vertices and 'E' is the number of edges.
